Hungotota is a 2D top-down roguelike shooter inspired by Brotato, built in Unity with FishNet multiplayer support.

You play as Hung — a fearless root-warrior — fighting waves of mutated potatoes in a pixel world gone wrong. Packed with unique mechanics and co-op action, this game offers more than just survival.

🌟 Key Features:
🧑‍🤝‍🧑 Up to 3-player online co-op via FishNet

🔫 Fast-paced twin-stick shooter gameplay

💥 Roguelike upgrade system: random weapons, stat boosts, perks

🧠 Smarter AI enemies with dynamic behavior

🎯 New gameplay mechanics beyond Brotato: dash, combo effects, skill synergy

🔁 High replayability with procedural waves and loadouts

🎨 Stylized 2D pixel art

🔧 Built With:
Unity (C#)

FishNet for multiplayer

Custom AI system

ScriptableObjects for scalable design

Pixel FX + Sound FX for juicy feedback
